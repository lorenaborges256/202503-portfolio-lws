This e-commerce mockup demonstrates my proficiency in creating a user-centric online shopping experience. The Weather App project showcases my skills in API integration and responsive design. It was developed with a mobile-first approach, ensuring smooth performance on smartphones and tablets, and highlights my ability to combine functionality with an elegant design.
